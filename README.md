# &lt;Lautaro /&gt; | Frontend Developer Portfolio

¡Hola! Bienvenido al repositorio de mi Portfolio Web Personal. 
Este proyecto fue desarrollado desde cero para mostrar mis habilidades técnicas, lógica de programación y proyectos más recientes en el área del Desarrollo Web Frontend.

##  Tecnologías Utilizadas

Este proyecto está construido sin frameworks externos para demostrar un dominio sólido de las bases de la web:

* **HTML5:** Estructura semántica y accesible.
* **CSS3:** Diseño Responsive (Mobile First), variables nativas, CSS Grid, Flexbox y animaciones personalizadas (keyframes).
* **JavaScript (Vanilla):** Manipulación del DOM, manejo de eventos y lógica de interfaz (como el menú desplegable interactivo).

##  Proyectos Destacados Incluidos

Dentro de este portfolio podrás encontrar enlaces al código fuente y a las demos en vivo de mis principales desarrollos:

1. **E-commerce Interactivo:** Plataforma de compras con carrito dinámico.
2. **Dashboard de Gastos:** Aplicación financiera con visualización de datos usando Chart.js.
3. **App del Clima:** Buscador meteorológico asíncrono consumiendo la API de OpenWeather.
4. **Buscador de Películas:** Catálogo interactivo conectado a la API de TMDB.
5. **Gestor de Usuarios:** Aplicación de lista de tareas centrada en la manipulación profunda del DOM.

##  Características Técnicas del Portfolio

* **Dark Mode Nativo:** Paleta de colores optimizada para la lectura prolongada.
* **Scroll Suave (Smooth Scrolling):** Navegación fluida entre las diferentes secciones mediante CSS nativo.
* **Animaciones UI/UX:** Efecto de máquina de escribir creado con CSS puro para el logotipo principal y transiciones suaves en componentes interactivos.
* **Totalmente Responsive:** Adaptable a dispositivos móviles, tablets y pantallas de escritorio.
